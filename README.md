📂 Unit 2: Linear Data Structures

This section dives into fundamental linear structures, built from scratch with a focus on dynamic memory handling and pointer-based logic.

🔗 Linked Lists
Singly Linked List (SLL)
Doubly Linked List (DLL)
Efficient implementations of:
Insertion (beginning, end, position-based)
Deletion (including edge cases like empty list, single node)
Clean handling of pointer updates to avoid memory inconsistencies

📚 Stack (SLL-Based)
Abstract Data Type (ADT) implementation using linked lists
Operations:
push()
pop()
peek()
Ensures O(1) time complexity for core operations

🚶 Queue (SLL-Based)
Linked list-based Queue implementation
Operations:
enqueue()
dequeue()
Maintains front and rear pointers for optimal performance
Achieves O(1) operations

🧠 Balanced Parentheses Checker
Stack-driven algorithm to validate expressions
Supports multiple bracket types:
(), {}, []
Efficient parsing with real-time mismatch detection

⚡ Dynamic Array (Custom Implementation)
Built using Python’s ctypes module
Simulates low-level array behavior:
Manual resizing
Capacity doubling strategy
Demonstrates how high-level lists work under the hood

Language: Python 3.x
